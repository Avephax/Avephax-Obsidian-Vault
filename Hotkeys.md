	- Ctrl + Alt + ↕ - Move list and sublists up or down
	- Alt + T - **Timestamp** a Note/Card
	- Ctrl + ↕ = **Move** a bullet point's position
	- Ctrl + Alt + C = **Turn** Page into Hover Tab (Hover Editor)
